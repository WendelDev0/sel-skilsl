# Selva Club Agent Skills

Este repositorio contem skills compartilhadas da equipe Selva Club.

Quando o usuario pedir trabalhos de copy, anuncios, criativos, landing pages, emails, WhatsApp, revisao de mensagem comercial, oferta, posicionamento ou angulos de campanha da Selva Club, use `skills/copy-selva/SKILL.md`.

Quando o usuario pedir trabalhos de trafego pago, campanhas, Meta Ads, Google Ads, funil, publico, verba, CAC, ROAS, CPA, CTR, CPC, criativos de performance ou diagnostico de conta da Selva Club, use `skills/trafego-selva/SKILL.md`.

Sempre responda em portugues do Brasil, a menos que o usuario peca outro idioma.
