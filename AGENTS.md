# Agent Skills

Este repositorio contem skills compartilhadas da equipe Selva.

Quando o usuario pedir trabalhos de copy, anuncios, criativos, landing pages, emails, WhatsApp ou revisao de mensagem comercial, use `skills/copy-selva/SKILL.md`.

Quando o usuario pedir trabalhos de trafego pago, campanhas, Meta Ads, Google Ads, funil, publico, verba, CAC, ROAS, CPA, CTR, CPC, criativos de performance ou diagnostico de conta, use `skills/trafego-selva/SKILL.md`.

Sempre responda em portugues do Brasil, a menos que o usuario peca outro idioma.
